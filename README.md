This is the Official Repository for the Official Website for the Coding and Graphics Club. It is currently using XAMPP as a development server.

USAGE 🔁:

Import cgc_dbase.sql.
Extract all the files from this repo to the XAMPP htdocs folder.
Run "localhost/CGC/home.php" to start off.
